# Projeto Alan Turing - Pioneiro da Computação

Página web biográfica e institucional dedicada a homenagear o legado de Alan Turing, destacando suas contribuições fundamentais como matemático, herói de guerra e pai da ciência da computação.

O projeto foi desenvolvido como atividade prática e pedagógica junto aos alunos do 3º Ano A da escola EE Professor Doutor Geraldo Campos Moreira.

---

## Objetivos do Projeto
* Consolidar o aprendizado em desenvolvimento web front-end básico.
* Explorar a estruturação de conteúdo e semântica.
* Praticar estilização e alinhamento visual avançado.
* Discutir a importância histórica dos algoritmos e da Inteligência Artificial.

## Tecnologias Utilizadas
* HTML5: Estruturação semântica do conteúdo.
* CSS3: Estilização, responsividade e alinhamento visual.

## Como Visualizar o Projeto
1. Faça o clone do repositório:
   ```bash
   git clone https://github.com
   ```
2. Navegue até a pasta do projeto.
3. Abra o arquivo index.html em qualquer navegador web.
